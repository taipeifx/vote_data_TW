Daniel Chen NYC Data Science Academy

For my first project I decided to do an analysis on Taiwan presidential election statistics for the six elections that have taken place since Taiwan held the first election in 1996.

The raw data I collected from ‘https://data.gov.tw/’,‘https://data.gov.tw/’ contained 5 csv files for each year, the files were named elbase, elcand, elctks, elpaty, and elprof. They each contain specific information regarding each election, from district codes and candidate information to ballot totals and voting statistics. The files came without column names and were only decipherable by finding the key from GitHub user MISNUK at https://github.com/MISNUK/CECDataSet/blob/master/RawData_Format.md .

I used mySQL and R to clean and combine the data to readable tables, and then turned the data into chart, maps, and statistical information.

All my work is stored at the GitHub address https://github.com/taipeifx/vote_data_TW .

I then performed a Two Sample t-test (below) to see if the average vote was the same for both political parties or if the difference was statistically significant. The results came back with a p-value of 0.7607, which is insufficient to reject the null hypothesis: Over the years the average vote for both parties have been similar.

Part 4: Chi-Squared test and Chi-Squared Residuals Plot

Finally, I did the Chi-squared test which evaluates whether there is a significant association between the categories of two variables (district and year): #Null hypothesis (H0): the row and the column variables of the contingency table are independent. #Alternative hypothesis (H1): row and column variables are dependent

library(tidyr)
kmtchisqdata = kmt[c(-3,-4,-5,-7,-8,-9)] 
kmtchisqdata = spread(data = kmtchisqdata, key = ANameE, value = Perc)
#Next, because some districts were renamed in 2012, we need to fill in some missing data:
kmtchisqdata[10][5:6,]= kmtchisqdata[9][5:6,]
kmtchisqdata[15][1:4,]= kmtchisqdata[23][1:4,]
kmtchisqdata[23][5:6,]= kmtchisqdata[15][5:6,]
kmtchisqdata[19][5:6,]= kmtchisqdata[18][5:6,]
kmtchisqdata[21][5:6,]= kmtchisqdata[20][5:6,]
kmtchisqdata = kmtchisqdata[-23] #Taipei County renamed to New Taipei City

chisq = chisq.test(kmtchisqdata[-1])
chisq
## 
##  Pearson's Chi-squared test
## 
## data:  kmtchisqdata[-1]
## X-squared = 347.42, df = 120, p-value < 2.2e-16

With a p-value < 2.2e-16, we can reject the null hypothesis and conclude that votes by district and votes by year are dependent.

library(corrplot)
par(mfrow=c(1,1))
corrplot(chisq$residuals, is.cor = FALSE)
